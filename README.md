## 📌 Overview  
Estate is a modern real estate landing page built using React, designed to present property listings with a clean, elegant, and conversion-focused UI.  
It emphasizes visual hierarchy, smooth user experience, and responsive design for all devices.

<p align="center">
  <img src="https://github.com/aliiashraf777/1rtld-estate-landing-page/blob/main/src/assets/15pr15preview.png" />
</p>

## 🎯 Problem  
- ❌ Traditional real estate websites lack visual appeal  
- ❌ Poor mobile responsiveness  
- ❌ Weak property presentation flow  
- ❌ Outdated UI/UX structure

- ✅ Modern and clean landing page design
